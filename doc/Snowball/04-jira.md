# Jira
[链接](http://docs.snowballfinance.com/pages/viewpage.action?pageId=3990330)

## Hierarchy
    story --> task --> bug

    story       product feature
    task        assigned to different team for development, front-end/back-end/design, etc
    bug         specific issue

## Mandatory info
    - type
    - description
    - bug owner
    - epic (function type)
    - sprint
    - version

## Granularity
    1-2 days, 5 days maximum
